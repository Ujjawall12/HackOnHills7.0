# Manhattan App

A Flutter mobile application for secure data erasure operations. Part of the Manhattan project for HackOnHills7.0.

## Features

- Secure data erasure for multiple device types
- User authentication and management
- Cross-platform support (Android, iOS, Web, Desktop)

## Getting Started

This Flutter application provides a mobile interface for Manhattan's secure data erasure capabilities.

### Prerequisites

- Flutter SDK (3.7.2 or higher)
- Dart SDK
- For Android development: Android Studio
- For iOS development: Xcode (macOS only)

### Installation

```bash
cd Manhattan/Manhattan_App
flutter pub get
```

### Running the App

```bash
# Android
flutter run -d android

# iOS
flutter run -d ios

# Web
flutter run -d chrome

# Windows
flutter run -d windows

# Linux
flutter run -d linux

# macOS
flutter run -d macos
```

## Dependencies

- `flutter_secure_storage`: Secure local storage
- `http`: HTTP requests
- `fluttertoast`: Toast notifications
- `intl`: Internationalization
